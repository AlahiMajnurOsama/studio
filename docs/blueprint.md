# **App Name**: ChromaShop

## Core Features:

- Product Browsing: Browse products by category or search.
- Product Details: Display product details, including images, description, and price.
- Product Sorting: Sort products by price, popularity, and other relevant factors.
- AI Product Recommendations: Provide product recommendations based on browsing history using an AI tool to intelligently recommend appropriate products.
- Color filtering: Interactive filtering of the color to browse through different colors available.
- Customizable product list: Allows customer to sort items into customizable favorites or wishlist. Product data will be handled using Firebase Firestore and Auth.

## Style Guidelines:

- Primary color: Saturated Purple (#9D4EDD) for a vibrant and modern feel. This is an expression of uniqueness in an otherwise drab world of eCommerce.
- Background color: Light Lavender (#E9D5FF), a desaturated version of the primary color for a soft and calming backdrop.
- Accent color: Electric Blue (#5A86FF) to create contrast, call attention to important elements, and help provide a consistent colorful feeling.
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern and clean aesthetic.
- Use modern and minimalist icons throughout the app.
- Subtle transitions and animations to enhance user experience. All button and navigational items will have a very minimal 'pulse' animation to signal feedback.
- Clean and intuitive layout with clear visual hierarchy.
